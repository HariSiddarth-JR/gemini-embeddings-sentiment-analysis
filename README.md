# Gemini Embeddings Sentiment Analysis

## Overview

This project performs sentiment classification on tweets using
Google Gemini Embeddings and XGBoost.

## Features

- Text preprocessing
- Gemini Embedding Generation
- XGBoost Classification
- Exploratory Data Analysis
- Confusion Matrix
- Feature Importance Analysis
- Custom Tweet Predictions
- Word Cloud Visualizations

## Technologies Used

- Python
- Google Gemini API
- XGBoost
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Results

The model successfully classifies tweets into:

- Positive
- Negative
- Neutral
